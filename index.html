Drag and drop visual editor

The easy-to-use visual editor doesn't require any programming knowledge. If you can use Microsoft Word and Excel, you can build apps.

Customizable and extensible

Control the look and feel of your apps with your own HTML and CSS code. Write complex business logic with custom JavaScript code, import any library.

Easy database management

Bricks is especially optimized for building database-driven, CRUD web apps. All the features you'll need in your apps are one click away.

Open technologies

Bricks is built on open standards, open source code and an open community. You can easily host and run your apps anywhere.
